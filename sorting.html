Sorting Algorithms

Sorting algorithms arrange elements in a particular order.

Sorting Basics

In-place sorting uses a constant amount of space to sort the inputs.
Internal sorting sorts data enntirely in main memory.
External sorting sorts large data by processing in smaller parts, often using disk storage.
Stable sorting keeps the original order of equal value elements.
Comparison-based sorting compare elements.
Non-comparison-based sorting does not compare elements.
Hybrid sorting mixes different sorting algorithms to enhance performance.
