Data Structures

What is data?

Data is defined as a value or set of values. For example: a name, a number, or a date are all data.

Variables

A variable is a named storage location in programming that holds data which can be changed or manipulated during the execution of a program. Variables can either be dynamically or statically typed. For dynamically typed variable, the type does not need to be declared upon creating the variable, and the type can be changed. For statically typed variables, a type must be declared and cannot be later changed.

Data Types

Data types are classifications that define the kind of data a variable can hold, such as integers, floating-point numbers, strings, or booleans. Some common data types include:

Boolean: true or false
String: a sequence of characters. Usually defined by double or single quotes. For example: 'I am 19 years old', '5', and 'false' are all strings.
Number: numeric values. Most programming languages further classify numbers as either ints (integers) or floats (numbers containing decimals).
Integer: whole numbers, such as 1, 3, 19, 100, etc.
Float: aka 'floating point' numbers. Numbers which contain decimals, such as 3.0, 43.2, or 100.1.
Character: a single character, such as !, a, or a blank space.
Date: a calendar date of form YYY-MM-DD.

Data Structures

A data structure is a method for organizing and storing data in a computer so that it can be accessed and used efficiently. It provides a way to manage large amounts of information by enabling operations like searching, sorting, and modifying data with ease. Common data structures include arrays, linked lists, stacks, queues, trees, and graphs, each designed to handle specific types of tasks and data relationships.

Memory

Memory in computing refers to the hardware components that store data and instructions for processing by a computer. It is primarily categorized into:
Primary Memory (RAM): Fast, volatile storage used by the CPU to hold data and instructions currently in use. Data in RAM is lost when the power is turned off.
Secondary Memory (Storage): Non-volatile storage like hard drives or SSDs that retain data even when the computer is powered down, used for long-term data storage.

Memory Management

Memory management is the process by which an operating system or a program efficiently allocates, tracks, and deallocates memory resources to ensure optimal performance and prevent issues like memory leaks or fragmentation. Key aspects include:

Allocation: Assigning memory space to programs and data when needed. This can be done statically (fixed size at compile-time) or dynamically (adjustable size at runtime).
Deallocation: Releasing memory that is no longer in use, making it available for future allocations.
Garbage Collection: An automatic memory management feature in some programming languages that identifies and frees memory that is no longer referenced by the program.
Memory Hierarchy Management: Optimizing the use of different types of memory (cache, RAM, storage) to balance speed and capacity for efficient data access.
