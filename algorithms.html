Introduction to Algorithms

An algorithm is a defined set of steps for performing a task. Algorithms take in data as an input and return an output after a series of steps.

Deterministic vs Non-Deterministic Algorithms

Deterministic algorithms produce the same output every time given the same input. You can imagine this as a straight line path from a specific input to its respective output.
Non-Deterministic algorithms may produce different outputs for the same input. Often used in scenarios involving randomness or parallel processing. You can visualize this as a trail with many diverging paths.

Efficiency

Time complexity and space complexity are two important considerations when implementing algorithms.Time complexity measures how running time of an algorithm increases with size of input. Space complexity assesses the amount of memory an algorithm uses relative to the input size.
